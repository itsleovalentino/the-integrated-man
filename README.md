# The Integrated Man — Daily

A calm, faith-driven daily dashboard for men: scripture, a journaling prompt, prayer list,
keystone habits, an end-of-day reflection, a weekly seven-pillar check with an Integration
score and trends, an in-app Bible reader with anchor verses, a full paper-style journal,
and floating thought-capture (type or voice) with memories and a week-in-review.

Single-file web app — vanilla HTML/CSS/JS, no build step, no backend. Each person's data
lives privately in their own browser (localStorage). Works offline once installed.

## Run / host
It's just `index.html`. Open it directly, or host it (GitHub Pages, Netlify, etc.) to get a
shareable https link — which also enables the microphone (voice) and notification features.

- `index.html` — the entire app
- `sw.js` — service worker for offline support

## Install on a phone
Open the hosted link in Safari/Chrome → Share → **Add to Home Screen**. It installs as
"The Integrated Man" and opens full-screen.
